# ***车辆详情共分三个子页面，接口分开处理，本文档的请求参数均为车辆的vin***
> /pages/car-detail

```
  params: {
    vin: 123123123
  }
```

### 车辆详情-实时
  + 获取车辆最后一次定位的时间和位置
  ```
  return {
    position: {
      location: '北京市朝阳区花园路附近',
      timesBefore: 123123123123   // 时间戳
    }
  }
  ```
