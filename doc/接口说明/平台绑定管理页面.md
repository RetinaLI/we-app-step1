
# 小程序-平台绑定页面
> author： 周维明
>
> /pages/profile/platform

### 1. 获取当前用户绑定的平台列表

  ```
    params: {

    }

    return: {
      list: [
        {
          唯一标识: 12,
          企业code: 'E017',
          用户昵称: '章三',
          登录账户名: 'sa',
          企业名称: '欧曼车联网平台'
        },...
      ]
    }
  ```

### 2. 解除指定平台的绑定

  ```
    params: {
      唯一标识: 12
    }
    return: {
      成功 or 失败
    }
  ```

### 3. 绑定新平台

  ```
    params: {
      企业code: 'E016',
      账户名: 'sa',
      密码: '123123'
    }
    return: {
      成功 or 失败
    }
  ```

### 4. 登录到指定平台

  ```
    params: {
      企业code: 'E016',
      账户名: 'sa'
    }
    return: {
      成功 or 失败
    }
  ```
